# ToDoListKnockoutJS
ToDo list portable app, uses KnockoutJS, BS, HTML, CSS and localStorage

# DEMO

[See demo](http://todolistknockoutjs.13signs.pl)

# INSTALLATION

To install just copy folder and open index.html in modern browser.
